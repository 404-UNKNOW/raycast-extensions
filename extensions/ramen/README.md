# Ramen Dashboard

Track your journey to Ramen Profitability. Real-time revenue in your menu bar.

## Features

- **Real-time Revenue**: Monitor your daily MRR and sales directly from the macOS menu bar.
- **Visual Progress**: Dynamic ramen bowl icon that evolves as you reach your goals.
- **7-Day Trends**: Unicode sparklines to visualize your recent growth.
- **Privacy Mode**: Quickly hide your revenue amounts with a keyboard shortcut.
- **Quick Share**: Share your milestones directly to X (Twitter).
- **Multi-Platform**: Support for both Stripe and Lemon Squeezy.

## Installation

This is a Raycast extension. You can install it from the Raycast Store.

## Configuration

To use this extension, you will need:
- A Stripe Restricted API Key (Read-only access to balance transactions).
- (Optional) A Lemon Squeezy API Key.
